Spring in Action,Fourth Edition is a hands-on guide to the Spring Framework, updated for version 4.
It covers the latest features,tools,and practices including Spring MVC,REST,Security, Web Flow,
and more.You'll move between short snippets and an ongoing example as you learn to build simple
and efficient J2EE applications. Author Craig Walls has a special knack for crisp and entertaining
examples that zoom in on the features and techniques you really need.
